# 大学生AI职业规划平台 - 技术方案文档

## 一、系统概述

### 1.1 项目背景
本项目针对大学生就业难、择业迷茫的痛点，构建了一个基于大模型的智能职业规划平台。平台提供考研/就业/考公决策分析、简历诊断、岗位推荐三大核心功能，帮助大学生明确职业方向、提升求职竞争力。

### 1.2 目标用户
- 计算机及相关专业大学生
- 对职业规划有需求但缺乏专业指导的在校学生
- 需要简历优化和岗位匹配服务的学生群体

### 1.3 技术选型
| 组件 | 技术选型 | 说明 |
|------|---------|------|
| 前端框架 | Streamlit | 快速构建交互式Web应用 |
| 大模型 | GLM-5.1（智谱AI） | 提供对话和生成能力 |
| 提示词管理 | YAML | 结构化提示词配置 |
| 向量数据库 | 内存向量库（FAISS风格） | 轻量级RAG实现 |
| Embedding | sentence-transformers | 文本向量化 |
| Python版本 | 3.10+ | 兼容性良好 |

### 2.2 数据流向

#### 2.2.1 决策辅助流程
```
用户输入（专业/成绩/兴趣/诉求）
    ↓
Planner Agent 分析意图
    ↓
Executor Agent 调用 career_prompt
    ↓
GLM-5.1 生成分析结果
    ↓
返回 Markdown 格式报告
```

#### 2.2.2 简历诊断流程（RAG增强）
```
用户粘贴简历
    ↓
RAG 检索相似简历样本
    ↓
拼接参考上下文 + 原始简历
    ↓
Executor Agent 调用 resume_prompt
    ↓
GLM-5.1 生成诊断建议
    ↓
返回结构化诊断报告
```

#### 2.2.3 岗位推荐流程（RAG增强）
```
用户输入（技能/城市/薪资）
    ↓
RAG 检索匹配岗位
    ↓
拼接岗位参考 + 用户需求
    ↓
Executor Agent 调用 job_prompt
    ↓
GLM-5.1 生成推荐结果
    ↓
返回岗位列表+学习路径
```

## 三、核心模块设计

### 3.1 提示词工程模块

#### 3.1.1 提示词结构
每个提示词 YAML 文件包含：
- **role**: 角色定义
- **task**: 任务说明
- **output_format**: 输出格式约束
- **constraints**: 约束条件列表
- **examples**: few-shot 示例
- **steps**: 执行步骤指引

#### 3.1.2 提示词配置示例（career_prompt.yaml）
```yaml
role: "职业规划分析师"
task: |
  你是一名专业的大学生职业规划顾问，帮助用户分析考研、就业、考公三条路径...
output_format: "markdown"
constraints:
  - "分析必须结合用户具体背景"
  - "每个方向的优势和挑战各3-5条"
  - "优先级建议要明确排序并说明理由"
```

### 3.2 Agent 框架

#### 3.2.1 Planner Agent
- **职责**：分析用户自然语言输入，判断任务类型，提取关键参数
- **输出**：JSON 格式的任务分析结果
- **降级策略**：当 LLM 调用失败时，使用关键词匹配进行分类

#### 3.2.2 Executor Agent
- **职责**：根据任务类型调用对应的提示词配置，执行具体任务
- **支持任务**：career（职业规划）、resume（简历诊断）、job（岗位推荐）
- **扩展性**：预留工具调用接口，支持后续接入外部API

### 3.3 RAG 检索模块

#### 3.3.1 Embedder（向量化）
- 使用 sentence-transformers 库
- 支持单条和批量向量化
- 包含简单文本分块功能

#### 3.3.2 RetrievalEngine（检索）
- 基于余弦相似度计算
- 支持 top_k 检索
- 可配置相似度阈值过滤

## 五、部署说明

### 5.1 环境要求
- Python 3.10+
- Streamlit 1.28+
- OpenAI SDK 1.0+

### 5.2 安装依赖
```bash
pip install -r requirements.txt
```

### 5.3 配置API密钥
在 Streamlit Secrets 中配置：
```toml
GLM_API_KEY = "your_api_key_here"
```

### 5.4 启动应用
```bash
streamlit run app.py
```
